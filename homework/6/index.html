Part 1: At Least 4 Decimals. A form to get numeric input with at least 4 decimal places, e.g., 1.1234. If invalid input is entered an error is shown. Otherwise, the input is rounded, square rooted then rounded, and rounded to nearest tenth, hundredth, and thousandth position. These results are shown.

Part 2: Character Counter. A form that accepts a text input and a character to search. When the form is submitted, the character count is shown. If no character is found, a new window appears with an error message.

Part 3: Phone Number Format Validator. A form that validates input is a phone number with format (999) 999-9999. A regular expression is used to validate the user input.
